# OpenAI Study: ChatGPT

# Technologies
- Javascript
- React
- NodeJs
- Express
- OpenAI

# How to use
- Clone the repository
- Access the project directory
- Grant permission to the directory `chmod -R 777 ./`
- Add your token in `index.js`
- Run `npm install`
- Run `node index.js`
- Access the directory `client`
- Run `npm install`
- Run `npm start`
- Access `localhost:3000` in browser
